# INF512-Unidad1-POO
Este repositorio es el correspondiente a la Unidad 1 de la asignatura INF-512.

El proyecto presenta un ejemplo claro y sencillo de Programación Orientada a Objetos.
utilizando una clase la cual tiene por nombre Mascota.

En este se le dará uso a los conceptos como:
Clase,
Atributos,
Métodos.
